# Data Sync Automator

[![Sponsor](https://img.shields.io/badge/Sponsor-GitHub%20Sponsors-ea4aaa?logo=githubsponsors)](https://github.com/sponsors/sdotwinter)

CLI tool for key-based syncing between local CSV/JSON datasets.

## Usage
```bash
python3 main.py dry-run --source examples/source.json --target examples/target.json .
python3 main.py sync --source examples/source.json --target examples/target.json .
python3 main.py status .
python3 main.py history .
```
