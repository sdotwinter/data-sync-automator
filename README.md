# Data Sync Automator

[![Sponsor](https://img.shields.io/badge/Sponsor-GitHub%20Sponsors-ea4aaa?logo=githubsponsors)](https://github.com/sponsors/sdotwinter)

Automate recurring data sync between sources and targets safely.

## Usage
```bash
python3 main.py plan --config samples/jobs.yaml
python3 main.py run --config samples/jobs.yaml --job users_sync
python3 main.py status --job users_sync
python3 main.py retry --config samples/jobs.yaml --job users_sync
```
